### 發布
```
hexo cl

hexo g

hexo d //發布

hexo s //本機預覽
```

### Project上傳至github

```
git  branch

git add .

git commit -m "內容"

git branch -M main

git push origin main --force
```

### 下載Project至別台電腦
```
npm install hexo-cli -g

git clone -b main https://github.com/3618321s/Project.git
```
