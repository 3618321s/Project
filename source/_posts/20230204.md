快速清除控制項資料或指定控制項狀態
使用這個方法
減少程式碼行數

<!--more-->
技術不夠
控制項來湊
不過有時候控制項太多
不想要寫很多行指定控制項


### 清除控制項
```
private void CheckControlValue(params Control[] ctols)
    {        
        foreach (Control ctol in ctols)
        {
            string cType = ctol.GetType().Name;
            if (typeof(TextBox).Name == cType)
            {
                (ctol as TextBox).Text = string.Empty;
            }
            else if (typeof(DropDownList).Name == cType)
            {
                (ctol as DropDownList).SelectedIndex = -1;
            }
            else if (typeof(CheckBox).Name == cType)
            {
                (ctol as CheckBox).Checked = false;
            }
            else if (typeof(CheckBoxList).Name == cType)
            {
                (ctol as CheckBoxList).SelectedValue = "%";
            }
            else if (typeof(RadioButton).Name == cType)
            {
                (ctol as RadioButton).Checked = false;
            }
            else if (typeof(RadioButtonList).Name == cType)
            {
                (ctol as RadioButtonList).SelectedIndex = -1;
            }
            else if (typeof(Label).Name == cType)
            {
                (ctol as Label).Text = string.Empty;
            }
        }       
    }
```
### 指定控制項 不顯示 false / 顯示 true
```
private void CheckControlValue(params Control[] ctols)
    {        
        foreach (Control item in items)
            {
                string cType = item.GetType().Name;
                if (typeof(TextBox).Name == cType)
                {
                    (item as TextBox).Visible = false;
                }
                else if (typeof(DropDownList).Name == cType)
                {
                    (item as DropDownList).Visible = false;
                }
                else if (typeof(CheckBox).Name == cType)
                {
                    (item as CheckBox).Visible = false;
                }
                else if (typeof(CheckBoxList).Name == cType)
                {
                    (item as CheckBoxList).Visible = false;
                }
                else if (typeof(RadioButton).Name == cType)
                {
                    (item as RadioButton).Visible = false;
                }
                else if (typeof(RadioButtonList).Name == cType)
                {
                    (item as RadioButtonList).Visible = false;
                }
                else if (typeof(Label).Name == cType)
                {
                    (item as Label).Visible = false;
                }
            }       
    }
```
### 指定控制項 不啟用 false / 啟用 true
```
private void CheckControlValue(params Control[] ctols)
    {        
        foreach (Control item in items)
            {
                string cType = item.GetType().Name;
                if (typeof(TextBox).Name == cType)
                {
                    (item as TextBox).Enabled = false;
                }
                else if (typeof(DropDownList).Name == cType)
                {
                    (item as DropDownList).Enabled = false;
                }
                else if (typeof(CheckBox).Name == cType)
                {
                    (item as CheckBox).Enabled = false;
                }
                else if (typeof(CheckBoxList).Name == cType)
                {
                    (item as CheckBoxList).Enabled = false;
                }
                else if (typeof(RadioButton).Name == cType)
                {
                    (item as RadioButton).Enabled = false;
                }
                else if (typeof(RadioButtonList).Name == cType)
                {
                    (item as RadioButtonList).Enabled = false;
                }
                else if (typeof(Label).Name == cType)
                {
                    (item as Label).Enabled = false;
                }
            }      
    }
```
